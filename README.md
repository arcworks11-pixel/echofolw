# EchoFlow 🌊

**EchoFlow** は、リスニングとシャドーイングに特化した、スマートでミニマルな学習ツールです。

## ✨ 特徴

- **2レイヤー字幕**: 原文と翻訳を同時に表示。個別に表示/非表示も可能です。
- **簡単同期機能**: 音源を再生しながら Enter キーを押すだけで、字幕の開始時間を記録できます。
- **アンカー機能**: 重要なポイントをブックマークとして記録し、ワンタップでジャンプ。
- **進捗バー・ビジュアライザ**: 再生バー上でアンカーの位置をひと目で確認。
- **スタディノート**: 学習中の気づきをその場でメモ。
- **100% プライバシー保護**: 入力したテキストや音声ファイル、学習データはすべてブラウザ内(localStorage)にのみ保存され、サーバーへ送信されることはありません。

## 🚀 使い方

1. **メディアを選択**: 学習したい音声または動画ファイルを選択します。
2. **スクリプト入力**: [Edit] モードで、学習したい文章を入力します。
3. **時間を記録**: [Sync] モードで再生しながら Enter キーを押し、文章と音声を同期させます。
4. **学習開始**: [View] モードで、自動スクロールされる字幕を見ながらリスニング・シャドーイングを行います。

## 🛠 テクノロジー

- HTML5 / CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)
- LocalStorage (データ保存)
